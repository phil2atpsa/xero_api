Importing Invoices
Please follow the steps below:

1. Download the nvoice template file
Start by downloading the CSV (Comma Separated Values) template file. This file has the correct column headings needed to import your invoice data.
Download File

2. Copy your invoices in to the template
Export your invoices from your system as a Comma Separated Values separated list. Using Excel or another spreadsheet editor, copy and paste your invoices from the exported file into the template. Make sure the invoice data you copy matches the column headings provided in the template. IMPORTANT: Do not change the column headings provided in the template. These need to be unchanged for the import to work in the next step. Dates are assumed to be in English format. For example, 25-12-2018. And please note that Invoices with invoice number(TID) that are already in the system will be ignored. If Contact details not found the system will create with details provided.
